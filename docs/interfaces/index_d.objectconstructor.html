isDictionary

  • isDictionary(value: unknown): value is Dictionary
    • Defined in index.d.ts:1183

    Returns true if the specified value is a dictionary. This method is similar to isPlainObject, but it has another output TS type: instead of inferring of an output type the method always cast the type to a dictionary.

    example
    interface Foo {
  bar(): number;
}

function foo(val: number | Foo) {
  if (Object.isPlainObject(val)) {
    val.bar(); // All fine
  }

  if (Object.isDictionary(val)) {
    val.bar(); // Warning: object is of type unknown
  }
}

    Parameters

    • value: unknown

    Returns value is Dictionary
