    • Defined in index.d.ts:2229

    Returns a string representation of a number by the specified pattern. All pattern directives are based on native Intl.NumberFormat options:

    1. 'style'
    2. 'currency'
    3. 'currencyDisplay'

    There are aliases for all directives:

    1. '$' - {style: 'currency', currency: 'USD'}
    2. '$:${currency}' - {style: 'currency', currency}
    3. '$d:${currencyDisplay}' - {currencyDisplay}
    4. '%' - {style: 'percent'}
    5. '.' - {style: 'decimal'}
    example
     100.50.format('$', 'en-us') // '$100.50'
 100.50.format('$:EUR;$d:code', 'en-us') // 'EUR 100.50'

    Parameters

    • num: number
    • Optional pattern: undefined | string

      string pattern of the format:

      1. symbol ';' is used as a separator character for pattern directives, for example: '$;$d:code'
      2. symbol ':' is used for specifying a custom value for a pattern directive, for example: '$:RUB;$d:code'
    • Optional locale: CanArray<string>

    Returns string
