createsAsyncSemaphore

  • createsAsyncSemaphore<T>(cb: function, ...flags: string[]): function
    • Defined in src/core/event/index.ts:92

    Wraps a callback function into a new function that never calls the target until all specified flags are resolved. The function returns a new function that takes a string flag and resolves it. After all flags are resolved, the last function invokes the target function. If you try to invoke the function after the first time resolving, ii won't be executed.

    example
    const semaphore = createsAsyncSemaphore(() => {
  console.log('Boom!');
}, 'foo', 'bar');

semaphore('foo');
semaphore('bar'); // 'Boom!'

// Function already resolved, the target function isn't executed
semaphore();
